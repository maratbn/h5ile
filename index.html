Welcome to the h5ile JavaScript utility / helper library for HTML5 File API support website.

  • Background:

    HTML5 File API is a W3C standard that allows local file access via web-based JavaScript.

    The HTML5 File API reuses the same HTML form input tag of type 'file' previously available in HTML 4.01, but with which it could only be used to upload the file contents to a remote web server rather than directly read them locally.
